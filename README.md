# Fedora basic setup

Script for setup Fedora

## Basic setup

* Enable RPM Fusion Free.
* Enable RPM Fusion NonFree.
* Install Multimedia codec package.
* Install Gnome tweaks.
* Enable Flathub in Gnome software app.
